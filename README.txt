# 🌍 TerraCubes

TerraCubes is a 3D cubic world generation engine for Minecraft that replaces the traditional chunk system with a fully volumetric cube-based system.

It enables massive vertical terrain, deeper caves, and expanded world height limits.

---

## 🚀 Features

- True 3D cube-based world generation (16x16x16)
- Expanded vertical terrain (higher mountains & deeper caves)
- Advanced terrain generation possibilities
- Built for Minecraft Forge 1.20.1

---

## 🧩 Requirements

- Minecraft 1.20.1  
- Minecraft Forge (47+)  

---

## 📦 Installation (Players)

1. Install Minecraft Forge  
2. Place the TerraCubes `.jar` file into your `mods` folder  
3. Launch Minecraft  

---

## ⚠️ Important Notes

- This mod is experimental and may contain bugs  
- Not all mods are compatible with cubic world generation  
- Worlds generated with TerraCubes are NOT compatible with vanilla worlds  
- Always backup your saves before use  

---

## 🛠️ Development Setup (For Modders)

This project follows the standard Minecraft Forge MDK setup.

### IntelliJ IDEA
1. Import the project as a Gradle project  
2. Run:
   ./gradlew genIntellijRuns  
3. Refresh Gradle if needed  

### Eclipse
1. Run:
   ./gradlew genEclipseRuns  
2. Import as Existing Gradle Project  

### Useful Commands
gradlew --refresh-dependencies  
gradlew clean
